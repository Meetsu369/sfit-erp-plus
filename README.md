## 📖 About

SFIT ERP+ is a Flutter-based mobile application developed to provide students with a modern and seamless way to access their college ERP.

Instead of replacing the ERP backend, the application securely interacts with the existing ERP portal while offering a significantly improved user experience, faster navigation, and additional productivity features.

This project is **independent and open source** and is intended for educational purposes.
